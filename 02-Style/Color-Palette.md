# Color Palette

| Role | Name | Hex | Effect | Pinterest Use | Website Use |
|---|---|---|---|---|---|
| **Primary** | Deep Navy | `#1A2A3A` | Grounded, luxurious, architectural | Text boxes, strong overlays | Main headers, footer background |
| **Secondary** | Rich Ochre | `#C48B36` | Warm, artisanal, energetic | CTA buttons, highlight words | Buttons, active states |
| **Accent** | Terracotta | `#A95C45` | Earthy, cultural, rich | Thin borders, subtext | Category tags |
| **Background** | Warm Cream | `#F7F5F0` | Clean but not sterile, inviting | Pin backgrounds, collage bases | Main body background |
| **Text** | Soft Black | `#2C2C2C` | Readable, elegant | Pin body text | All paragraph text |
| **Soft Neutral** | Oatmeal | `#E5E0D8` | Subtle separation | Text box backings | Product card backgrounds |
| **Metallic / Luxury** | Brushed Brass | `#D4AF37` | Premium | Minimal accents | Small UI icons |

## Rules
- Always pair Navy text on Cream background for max contrast.
- Use Ochre **only** for CTAs and highlight moments — never for body text.
- Brushed Brass is reserved for icons and luxury accents. Never use as a fill.
- Background of all pins, by default: `#F7F5F0` Warm Cream.
