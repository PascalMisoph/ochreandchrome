# Canva Template System *(LEGACY REFERENCE)*

> **Status: Legacy.** Ochre & Chrome no longer uses Canva as the production composition tool. Pins are now generated as fully-composed designs (label + headline + subline + image + footer + domain) in a single ChatGPT Images v2 call. See [[AI-Prompts]] §7.A and [[Visual-Direction]] for the live workflow.
>
> This document is **retained as type, palette, and layout reference**. The 12 layout patterns below describe pin *concepts* (Product Roundup, Best X for Y, Mistakes, How-To, Trend Forecast, Shop the Look, Moodboard, Comparison, Blog Teaser, Video Cover, Before/After, Checklist) that still inform how we frame `{HEADLINE}`, `{SUBLINE}`, `{ALIGNMENT}`, and `{IMAGE_BRIEF}` inside the §7.A master prompt. The rendered output, however, comes from ChatGPT Images v2 — not from Canva.
>
> Do not duplicate Canva templates. Do not export from Canva for production pins.
