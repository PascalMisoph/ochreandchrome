# 30-Day Pinterest Content Plan — 75 Pins

The operational plan. **Day 1 = first day of execution after this vault is read.** 75 pins across 30 days at 2.5 pins/day average (mostly 2; selected high-output days = 3).

> **Production model (mandatory):** Every pin is generated as a **fully-composed Pinterest design** via ChatGPT Images v2 using the master prompt formula in [[AI-Prompts]] §7.A. The image returned is publishable as-is — no Canva, no post-composition. Each pin entry below has an **MJ prompt** field — read it as the `{IMAGE_BRIEF}` fragment to drop into the §7.A master prompt, alongside the entry's `Title` (`{HEADLINE}`), `Overlay` (`{SUBLINE}`), board (drives `{CATEGORY_LABEL}`), and aesthetic world. Old "Canva Template X" notes in the per-pin **Notes** field are *legacy*; ignore them and use §7.A. See [[Visual-Direction]] and [[Pin-Quality-Checklist]].

## How to read each entry

Each pin block uses these 16 fields exactly:

```
P##  Day · Pillar · Board · Format
Title           — max 40 chars
Overlay         — on-image text
Description     — pin description
KW (primary)    — primary keyword
KWs (secondary) — 4 secondary keywords
Image direction — visual brief
MJ prompt       — Midjourney prompt
URL / dest      — destination
Save angle      — why a user saves
Difficulty      — easy / medium / high
Notes           — for the agent
```
