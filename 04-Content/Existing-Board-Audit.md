## Cross-board rules

- Every pin lives on **its primary topical board** + **AllPins** (secondary).
- Fusion content (Neo Deco × Afrohemian) goes to *both* core boards if both are clearly present, otherwise to the one that dominates.
- Trend-translation pins go to **2026 Home Decor Trends** primarily, plus the relevant aesthetic board if applicable.
- An Adire pin always lives on **Authentic Adire** primary, **Afrohemian Decor & Interiors** secondary, **AllPins** tertiary.
- Update each board description quarterly with refreshed long-tail keyword stacks.
