## Link Cloaking & Tracking

1. Install **ThirstyAffiliates** (or Lasso) on WordPress.
2. Cloak format: `ochreandchrome.com/recommends/cb2-brass-pendant`
3. Categorize links in the plugin **by room and aesthetic** so they're findable fast.
4. Add the FTC disclosure (italicized) to the **top** of every monetized post — see [[Site-Architecture]].

See [[Product-System]] for the selection checklist before adding any product.
