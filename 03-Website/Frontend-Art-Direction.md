# Frontend Art Direction

Purpose: project-specific initialization for the `image-taste-frontend` workflow.

## Brand Translation

Ochre & Chrome should feel like an editorial interiors publication with boutique-catalog discipline: warm, architectural, worldly, polished, and materially specific. The frontend should avoid sterile SaaS layouts, generic lifestyle-blog templates, and cluttered product collages.

Core promise to express visually:

> Intentional living. Curated spaces.

## Visual System

- **Palette:** Warm Cream `#F7F5F0` foundation, Deep Navy `#1A2A3A` for high-contrast text and footer surfaces, Rich Ochre `#C48B36` for primary CTAs, Terracotta `#A95C45` for tags and small editorial accents, Oatmeal `#E5E0D8` for quiet panels, Brushed Brass `#D4AF37` only as a restrained accent.
- **Typography:** Playfair Display for editorial headlines, Lato for body and supporting UI, Montserrat all-caps for short labels.
- **Imagery:** Full-width interiors, tactile material crops, product cutouts on clean backgrounds, marble veining, burl wood, sisal, Adire, velvet, brass, fluted glass. Images should live in controlled frames with 4px or 8px radius.
- **Spacing:** Generous negative space. No crowded edges. Sections should breathe like an expensive print layout.
- **Texture:** Use material contrast as the luxury signal, not glitter, neon, generic gradients, or excessive effects.

## Image-First Workflow

For visual frontend tasks, generate reference imagery before implementation, then analyze it as a design specification. Preserve:

- hero composition and headline scale
- image-to-text balance
- section cadence
- spacing rhythm
- color relationships
- media frame proportions
- CTA hierarchy
- editorial component language

The generated image is the visual source of truth. The implementation should translate it faithfully rather than drifting into a generic coded layout.

## Default Direction For Homepage Work

- **Theme paradigm:** Pristine Light Mode with editorial warmth.
- **Background character:** Quiet textured paper or material surface.
- **Typography character:** Elegant editorial serif plus clean sans.
- **Hero architecture:** Massive image-first hero with restrained text.
- **Section system:** Gallery-led visual cadence with editorial offsets.
- **Signature components:** Layered Image Crop Frames, Off-Grid Editorial Layout, Product UI Panel Stack, Oversized Metrics Strip.
- **Motion language:** Cinematic fade-through energy and parallax image drift energy.

## Homepage Structure To Respect

1. Hero with lead magnet opt-in
2. Shop by Aesthetic: Neo Deco / Afrohemian
3. Latest Styling Guides
4. The Curated Cart
5. Trend Report
6. Footer

Primary H1: **Design With Depth.**

Subheadline: Curated aesthetics, shoppable guides, and styling rules for the intentional modern home.

## Hard Avoids

- Cursive or script fonts
- Neon colors
- generic 3D-rendered rooms
- chaotic overlapping product collages
- text directly over busy imagery without a cream backing plate or contrast treatment
- generic startup phrasing or template-like card rows
- "boho-chic" terminology; use **Afrohemian**
