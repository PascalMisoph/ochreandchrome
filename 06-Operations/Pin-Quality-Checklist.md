# Pin Quality Checklist

Run this gate against **every pin** before scheduling. If any box fails, pull the pin and fix before shipping.

## Production-model gate (5 checks — pin-design integrity)

A pin does **not** pass the visual gate unless the **fully-composed image generated by ChatGPT Images v2** already contains all five of the following:

- [ ] **1. Readable integrated text.** The headline, category label, and footer text are *baked into the image*, sharp, mobile-readable, and free of generative artefacts. Not a plain interior photo to be captioned later.
- [ ] **2. Clear headline hierarchy.** Top category label (small, uppercase, tracked) → editorial serif headline (large, dominant) → optional subline (small, supporting). Hierarchy reads correctly at thumbnail scale.
- [ ] **3. Branded footer banner.** A solid warm-cream banner across the bottom containing the Ochre & Chrome wordmark in Playfair Display.
- [ ] **4. The domain `ochreandchrome.com`** is present in the footer banner, in clean tracked sans, deep navy or soft black on warm cream.
- [ ] **5. Brand-consistent editorial composition.** Palette and typography hold to [[Color-Palette]] and [[Typography]]; world identity holds to the relevant guide ([[Neo-Deco-Visual-Guide]] / [[Afrohemian-Visual-Guide]] / [[Adire-Textile-Visual-Guide]] / [[Brass-Lighting-Visual-Guide]]); the pin reads as Ochre & Chrome at first glance.

If any of these five fail, regenerate via [[AI-Prompts]] §7.A. Do not ship a pin that needs Canva touch-up to pass — that's a sign the prompt was wrong, not the pipeline.

## Metadata gate (12 checks — Pinterest performance & brand)

### 1. Search & metadata
- [ ] **Title is searchable.** Front-loaded primary keyword. ≤40 characters or accepts truncation gracefully.
- [ ] **Description fills 300+ characters.** Natural sentence first, 3 long-tail keywords woven in, specific CTA, 2–3 on-brand hashtags.
- [ ] **Destination URL exists and returns 200.** Not a 404. Not "Coming Soon."
- [ ] **Alt text is set.** Mirrors the description's first sentence.

### 2. Visual & format
- [ ] **2:3 aspect ratio at 1000 × 1500 px.** Not square, not 1:1, not 4:5.
- [ ] **Mobile-readable text.** Title overlay legible at thumb-thumbnail size. No 12-point body copy as overlay.
- [ ] **Brand pattern recognisable.** Cream `#F7F5F0` base or warm-world background. Playfair Display title. `ochreandchrome.com` watermark in Montserrat bottom-centre.
- [ ] **No clutter, no overlap.** Negative space at top and bottom. No edge-to-edge packing.

### 3. World & emotion
- [ ] **Clear world allegiance.** Pin reads as Neo Deco *or* Afrohemian *or* Adire *or* Brass Lighting *or* Trend Translation *or* explicit fusion. If you can't tell, the pin is muddled — re-shoot.
- [ ] **Save reason is obvious.** Inspiration, planning, or purchase intent — one of the three. Pin makes the user want to save it for *something specific*.

### 4. Cultural-care (apply only when relevant)
- [ ] **No "tribal," "ethnic," or "boho-chic" copy.** No invented cultural origin claims. No authenticity claims on AI-generated visuals. Adire correctly framed (Yoruba, Nigeria) when named. See [[Cultural-Care-Guidelines]].

### 5. Voice
- [ ] **Voice rules followed.** No "vibe," "obsessed," "cute," "hack," "bling," "you guys," "link in bio." Material-led, structurally specific, slightly opinionated. See [[Brand-Voice]].

## Optional bonus checks (for high-difficulty pins)

- [ ] Pin links to a destination that *answers the pin's promise* (no clickbait drift).
- [ ] If product roundup: every product passes the [[Product-System]] selection checklist.
- [ ] If comparison: both sides labelled clearly, no straw-man bias.
- [ ] If cultural content: [[AI-Prompts]] §7.14 audit run and passed.
- [ ] If affiliate-linked: link cloaked through ThirstyAffiliates, not raw.

## Common failure modes

| Failure | What it usually means | Fix |
|---|---|---|
| Pin is "pretty" but no one would save it | No clear save reason | Add specificity to the title (number, room, rule) |
| CTR < 0.5% after 7 days | Title and image disagree | Rewrite title to match image, or re-shoot image to match title |
| Saves high, outbound clicks zero | Destination doesn't deliver on the pin's promise | Redesign blog CTA buttons; add internal pin-promise paragraph at top of post |
| Description has duplicate copy across multiple pins | Pinterest deprioritises | Vary descriptions even when titles are similar |
| Pin reads as "boho" not Afrohemian | Visual is too pastel or too cluttered | Re-shoot per [[Afrohemian-Visual-Guide]]: warm afternoon light, layered but edited |
| Brass pendant pin doesn't read premium | Plated finish, daylight, no light pool | Re-shoot per [[Brass-Lighting-Visual-Guide]]: dusk, warm pool below, patina visible |

## When to override the checklist

Almost never. The two exceptions:

1. **A genuinely time-sensitive trend pin** where shipping a 90% pin today beats a 100% pin in three days. Override only #2 (description length) — never override #4 (cultural-care).
2. **A pure inspiration / brand-anchor pin** where the destination is the homepage (P31, P53). The "destination delivers on the promise" check is softer — but the URL must still return 200.
