# Start Here — For Agents

If you are an AI agent reading this vault for the first time, this is the file to read first. **10 minutes here saves 2 hours of guessing later.**

## What Ochre & Chrome is (one paragraph)

Ochre & Chrome is a Pinterest-first, editorial interior design publication targeting the 2026 **Neo Deco × Afrohemian** aesthetic intersection. The brand line is **"Design With Depth."** The promise is curated aesthetics, shoppable guides, and styling rules for the intentional modern home. Revenue model is affiliate (CB2, Lulu & Georgia, 54kibo, Etsy, Wayfair, Lumens). Audience is Gen X / Millennial homeowners renovating or redecorating, fatigued by fast furniture and sterile minimalism.

## The two worlds (do not collapse them)

The brand stands on two equal aesthetic worlds:

1. **Neo Deco** — geometry, glamour, polish. Brass, chrome, red marble, fluted glass, velvet, fan arches, modern Art Deco.
2. **Afrohemian** — soul, texture, craft. Linen, jute, indigo Adire textile, woven baskets, carved wood, terracotta, warm earth tones.

Neither dominates. Afrohemian is **not** a side trend — it carries roughly equal strategic weight to Neo Deco. See [[Neo-Deco-vs-Afrohemian]] for the full relationship and [[Cultural-Care-Guidelines]] for Afrohemian/Adire framing rules.

## How to generate a new pin

1. Open [[30-Day-Pinterest-Content-Plan]]. Find the next un-shipped pin in [[Pin-Production-Queue]].
2. Read the pin spec (16 fields — title, overlay, description, keywords, image direction, MJ prompt, board, etc.).
3. Use [[AI-Prompts]] §7.7–7.10 (world-specific image prompts) to refine the Midjourney prompt if needed.
4. Generate the image. Drop into Canva using the template named in the pin spec.
5. Apply title overlay and `ochreandchrome.com` watermark.
6. Run [[Pin-Quality-Checklist]] — every box must tick.
7. Upload to Pinterest with title, description, alt text, primary board (from spec), and AllPins as secondary.
8. Mark `[✓]` in [[Pin-Production-Queue]].

For brand-new pin concepts not in the plan, use [[AI-Prompts]] §7.11 (one-pin spec generator).

## What not to do

- Do **not** create new Pinterest boards. The six existing boards are the strategy.
- Do **not** use "boho-chic," "tribal," or "ethnic" language. See [[Cultural-Care-Guidelines]].
- Do **not** treat Afrohemian as a side trend. It's a flagship world.
- Do **not** invent cultural origin claims. Specificity over romanticisation.
- Do **not** cite Pinterest as the trend authority — frame trends as cultural/editorial movements.
- Do **not** use cursive/script fonts, neon colors, or generic 3D-rendered rooms.
- Do **not** create new top-level folders without checking with the user (per the rules in `CLAUDE.md` at the vault root).
- Do **not** confuse Adire (Yoruba, Nigeria) with mudcloth (Bamana, Mali) or kente (Akan, Ghana).
- Do **not** ship a pin that fails [[Pin-Quality-Checklist]].
- Do **not** post a pin to a 404 destination URL — write the blog post first.
