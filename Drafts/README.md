# Drafts

Working drafts of blog posts before they go live on `ochreandchrome.com/journal`.

## Convention

- Filename: `[day]-[slug].md` (matches the [[Blog-Plan-90d]] schedule)
- Each draft includes frontmatter-style metadata block (keyword, type, length, status, meta title, meta description)
- Sources at the bottom as markdown links
- No em dashes (house style)
- Phase 1 posts (first 3) ship without affiliate CTAs; Phase 2 posts insert cloaked links once CB2 / Lumens / Lulu & Georgia approve

## Workflow

1. Draft here with sources cited.
2. Review against [[Brand-Voice]] and [[Site-Architecture]] templates.
3. Once approved, paste into WordPress, add images, insert affiliate links via ThirstyAffiliates, publish to `/journal`.
4. Move file from `Drafts/` to `Drafts/Published/` (create on first publish) with the live URL added at the top.
